js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js，form.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS(js非模块化加载)
				
$(function(){
    $(".navbarlist").slidenavbar();
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，form/slidenavbar

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
				
require(['jquery','form/slidenavbar'], function($){
    $(function(){
        $(".navbarlist").slidenavbar();
    });
});

本插件须引用的css文件:

bootstrap.css，font-awesome.css，form.css


说明

可上下滑动及左右切换的导航栏一共分为上下两个部分，导航栏和其所要显示子菜单的内容。在导航栏中使用ue-slidenavbar 定义导航栏中的一个个按钮，使用data-target="#..."属性确定按钮所对应相对应部分的子菜单内容。

子菜单内容部分，要定义相应的容器.ue-slidenavbar-container和必要盛放切换内容的容器.ue-slidenavbar-list； .ue-slidenavbar-list容器里面定义相对应data-target的具体内容.ue-slidenavbar-item即可。
