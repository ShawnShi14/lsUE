popover右侧(失焦消失)
				
		
       $("#left").popover({
           title: "这是普通popover的标题",
           content: "定义标题普通popover的内容",
           placement: "left"
       });
       
       $("#top").popover({
           title: "",
           html: true,
           content: '
'+
                       ''+
                       ''+
                    '
',
           placement: "top"
       }); 
    
       $("#right").popover({
           placement: "right"
       });


需要引用的css文件：

bootstrap.css

需要引用的js文件：

jquery.js，bootstrap.js


说明


在使用失焦消失的popover弹出框时，为了更好的跨浏览器兼容，使用a标签，当使用a标签的时候一定要注意不要忘了tabindex属性。弹出框的位置使用js方法固定即可。

使用Popover组件有两种使用方法：一种是通过data-属性，一种是通过javascript方法调用。本例通过javascript来实现。


名称 类型 默认值 说明
animation boolean true 是否开启CSS动画
content string | function "" popover的内容,内容可以是普通文本，html代码，模板等，为html代码或者是模板的时候要设置html的值为true
title string | function "" 弹出框标题
html boolean false 向弹出框插入 HTML。如果设置为false时，内容中的html代码将会作为文本输出。如果设置为true时就会自动解析相应的html。
placement string|function top 规定如何定位弹出框（即 top|bottom|left|right|auto）。当指定为 auto 时，会动态调整弹出框。
trigger string 'click' 定义如何触发弹出框： click| hover | focus | manual。您可以传递多个触发器，每个触发器之间用空格分隔。
template string '<div class="popover" role="tooltip"><div class="arrow"></div><h3 class="popover-title"></h3><div class="popover-content"></div></div>' 定义模板，当html选项的值设为true时期作用。
delay number | object 0 延迟显示和隐藏弹出框的毫秒数 - 对 manual 手动触发类型不适用。如果提供的是一个数字，展现和消失的时候都会显示。提供对象的话结构如下: delay: { "show": 500, "hide": 100 }
container string | false false 将提示框放在特定的容器里
