• HTML
<select multiple="multiple" size="10" name="doublebox" class="demo"></select>
				
var demo2 = $('.demo').doublebox({
    nonSelectedListLabel: '选择角色',
    selectedListLabel: '授权用户角色',
    preserveSelectionOnMove: 'moved',
    moveOnSelect: false,
    nonSelectedList:[{"roleId":"1","roleName":"zhangsan"},{"roleId":"2","roleName":"lisi"},{"roleId":"3","roleName":"wangwu"}],
    selectedList:[{"roleId":"4","roleName":"zhangsan1"},{"roleId":"5","roleName":"lisi1"},{"roleId":"6","roleName":"wangwu1"}],
    optionValue:"roleId",
    optionText:"roleName",
    doubleMove:true
});

需要引用的css文件：

bootstrap.css，form.css，ui.css

需要引用的js文件：

bootstrap.js，form.js，ui.js

说明

在下拉框中如果想同时选择多个选项的时候，可以设置multiple属性， 还可以把 multiple 属性与size属性配合使用，来定义可见选项的数目。


名称 类型 默认值 说明
nonSelectedListLabel String,false false 指定非选择列表的名称
selectedListLabel String,false false 指定已选列表的名称
preserveSelectionOnMove String,false false 值可以是'all'（选择已移除的元素和被选中目标元素）或者是'moved'（只选择已移动的元素）
moveOnSelect boolean true 确定是否在选择上移动选项，在安卓浏览器上，这个选项是被选择的
nonSelectedList [ ] null 没被选择列表的内容
selectedList [ ] null 已选择列表的内容
optionValue String
optionText String
doubleMove boolean false 双击移动的功能
filterTextClear String 'show all' 文本为“全部显示”按钮
filterPlaceHolder String 'Filter' 为input添加占位符
moveSelectedLabel String 'Move selected' lable标签为移动所选选项按钮
moveAllLabel String 'Move all' lable标签为移动所有选项的按钮
removeSelectedLabel String 'Remove selected' lable标签为“删除”已选选项的按钮
removeAllLabel String 'Remove all' lable标签为“删除”所有选项的按钮
selectorMinimalHeight number 100 双列表的最小高度
showFilterInputs boolean true 是否显示
nonSelectedFilter String ' ' 一个非选择元件滤波器初始化双列表。这是只适用于如果showfilterinputs是真
selectedFilter String ' ' 一个已选择元件滤波器初始化双列表。这是只适用于如果showfilterinputs是真
filterOnValues boolean false
