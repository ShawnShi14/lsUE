js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js，bootstrap.ja，form.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS(js非模块化加载)
					
$(document).ready(function() {
    $("#txtName").transfer({
        target: "#txtCharsName",
        fullcharts: false
    });		
    $("#txtName").transfer({
        target: "#txtFullName",
        callback: function(target, value) {
            $(target).val(value.replace("/",""));
        }
    });		
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，form/pinyin

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
					
require(['jquery','form/pinyin'], function($){
    $(document).ready(function() {
        $("#txtName").transfer({
            target: "#txtCharsName",
            fullcharts: false
        });		
        $("#txtName").transfer({
            target: "#txtFullName",
            callback: function(target, value) {
                $(target).val(value.replace("/",""));
            }
        });		
    });
});

参数说明

名称 类型 默认值 说明
target String Null 获取拼音目标input文本框地址
fullcharts Boolean true 是否获取全拼，选项值为false时，获得简拼
evt String "blur" 触发方法事件
