arttemplate介绍

artTemplate 是新一代 javascript 模板引擎，在页面与数据分离工作中起重要作用。它采用预编译方式让性能有了质的飞跃， 调试功能也值得一提。模板调试器可以精确定位到引发渲染错误的模板语句，让开发变得高效。

说明

arttemplate 有两个版本的模板语法：简洁语法和原生语法可。推荐使用简洁语法，语法简单实用，利于读写。本实例采用的是简洁语法。

简洁语法

 
{{if admin}}
    {{include 'admin_content'}}

    {{each list}}
        <div>{{$index}}. {{$value.user}}</div>
    {{/each}}
{{/if}}

原生语法

                      
<%if (admin){%>
    <%include('admin_content')%>

    <%for (var i=0;i<list.length;i++) {%>
        <div><%=i%>. <%=list[i].user%></div>
    <%}%>
<%}%>

快速理解

编写模板

使用一个type="text/html"的script标签存放模板：

渲染模板

var data = {
    title: '标签',
    list: ['文艺', '博客', '摄影', '电影', '民谣', '旅行', '吉他']
};
var html = template('test', data);
document.getElementById('content').innerHTML = html;

常用方法

template(id,data)

根据ID渲染模板，内容会根据document.getElementById(id)查找模板，如果没有data参数，会返回渲染函数。

template.compile(sourceoptions)

将会返回渲染函数

template.rander(source,options)

将会返回渲染结果

template.helper(name,callback)

添加辅助方法

template.config(name, value)

更改引擎的默认配置。

字段 类型 默认值 说明
openTag String '{{' 逻辑语法开始标签
closeTag String "}}" 逻辑语法结束标签
escape Boolean true 是否编码输出 HTML 字符
cache Boolean true 是否开启缓存（依赖 options 的 filename 字段）
compress Boolean false 是否压缩 HTML 多余空白字符

<head>

    <script id="test" type="text/html">
        
           {{if isAdmin}}
             <h1>{{title}}</h1>

               {{each list as value i}}
                   <li>索引 {{i + 1}} ：{{value}}</li>
                {{/each}}
              </ul>
           {{/if}}  
    </script>
    <script type="text/javascript">
	
		 var data = {

		 list: ['文艺', '博客', '摄影', '电影', '民谣', '旅行', '吉他']
		  };

   </script>
</body>
</html>
