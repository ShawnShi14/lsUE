$(function(){
    //初始化
    var initPagination = function() {
        $(".pagination").pagination(20, {
            num_edge_entries: 1, 
            num_display_entries: 4,            
            items_per_page:1
        });
     }();
});

本插件须引用的css文件:

form.css

本插件须引用的js文件:

jquery.js form.js


说明

使用分页组件只需添加样式为pagination的DOM结构即可。


名称 类型 默认值 说明
num_edge_entriess number 0 两侧显示的首尾分页的条目数
num_display_entries number 10 连续分页主体部分显示的分页条目数
items_per_page number 10 每页显示的条目数
maxentries number 总条目数
current_page number 0 当前选中的页面，其中值为0时，表示第1页
link_to String # 分页的链接
prev_text String Prev “前一页”分页按钮上显示的文字
next_text String Next “下一页”分页按钮上显示的文字
ellipse_text String ... 省略的页数用什么文字表示
prev_show_always Boolean true 是否显示“前一页”分页按钮
next_show_always Boolean true 是否显示“下一页”分页按钮
callback 回调函数
