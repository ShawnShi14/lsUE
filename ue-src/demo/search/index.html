js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js，bootstrap.js，form.js，ui.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS(js非模块化加载)
$(function(){
    $("#query").search(function(value){
        alert(value);
    });
    		
    $(document).on("click","#query",function(){
        JPlaceHolder.init();
    });
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，collapse/collapse

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
			
require(['jquery','search/search','demo/prettify'], function($){
    $(function(){
        $("#query").search(function(value){
            alert(value);
        });
    		
        $(document).on("click","#query",function(){
            JPlaceHolder.init();
        });
    });
});

需要引用的css文件：

bootstrap.css，ui.css，font-awesome.css


说明

用户只需要定义一个按钮并绑定所提供的接口即可使用该组件。
