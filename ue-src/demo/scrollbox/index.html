js非模块化加载：

  • JS(js非模块化加载)
            	
$('#demo1').scrollbox({	
   linear: true,
   step: 1,
   delay: 0,
   speed: 100
});

js模块化加载：

  • JS(js模块化加载)
				
require(['jquery','form/scrollbox'], function($){
    $(function(){
        $('#demo1').scrollbox({	
           linear: true,
           step: 1,
           delay: 0,
           speed: 100
        });	
    });
});

说明


非模块化加载时需要引用的js文件：

jquery.js，form.js


模块化加载时在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，form/scrollbox

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

样式属性与值 所属样式 说明
overflow:hidden .scroll-text 只显示需要显示的内容
height .scroll-text 包裹滚动信息容器高度，当取值大于子<ul>标签外加一个孙子<li>标签的高度时，滚动效果无法实现。

js非模块化加载：

  • JS(js非模块化加载)
$('#demo3').scrollbox({
    direction: 'horizontal',
    distance: 134
});
$('#demo3-backward').click(function () {
    $('#demo3').trigger('backward');
});
$('#demo3-forward').click(function () {
    $('#demo3').trigger('forward');
});

js模块化加载：

  • JS(js模块化加载)
					
require(['jquery','form/scrollbox'], function($){
    $(function(){
        $('#demo3').scrollbox({
            direction: 'horizontal',
            distance: 134
        });
        
        $(document).on("click", "#demo3-backward", function(){
            $('#demo3').trigger('backward');
        });
        
        $(document).on("click", "#demo3-forward", function(){
            $('#demo3').trigger('forward');
        });
    });
});

说明


为了保证滚动的流畅性，如果需要同时显示 X 张图片时，请多留出1张不同的图片做替换，即在DOM结构上至少写 X + 1 个<li>标签。

如展示效果所示，方框中一共显示 5 张图片，但是在DOM结构中有 6 个<li>标签。


样式属性与值 所属样式 说明
overflow:hidden .scroll-img 只显示需要显示的内容
font-size: 0 .scroll-img 去除inline-block元素之间的空隙
display: width .scroll-img ul ul标签宽度，为了让图片滚动能正常运行，如果需要展示 X 张图片，则此值必须大于 X + 1<li>标签的宽度之和。
此值建议往多里写。
display: inline-block .scroll-img ul li 使li标签内容显示在同一行内

说明

本展示效果的HTML中的DOM结构以及引用的JS文件同上。

名称 类型 默认值 说明
linear Boolean false 滚动方式，是否是线性滚动。
step Number 5 每次滚动的像素值，只有当linear选项值为false时，选项起有用。
delay Number 2 每次滚动完毕后延迟几秒。
speed Number 32 每一步完成后延迟几毫秒。
direction String 'vertical' 滚动方向，选择'horizontal'的方向为上。
switchItems Number 1 每次滚动事件后切换的项目，当direction取值为‘horizontal’时使用。
注意：当需要将整排图片集体向左或向上翻滚时，整排显示了多少张图片，此值就设置为多少。
distance String/Number 'auto' 每次翻滚距离，图片翻滚时使用。
