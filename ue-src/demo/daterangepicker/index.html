$(document).ready(function() {
    $('#reservation').daterangepicker({
        format:'MM/DD/YYYY'
    },
    function(start,end){
         //输出ISO标准格式的时间与日期
        console.log(start.toISOString(),end.toISOString());
    });
});

需要引用的css文件：

bootstrap.css，ui.css，form.css

需要引用的js文件：

jquery.js，bootstrap.js，form.js，ui.js


说明

将内容放在.form-group中，避免结构混乱。

名称 类型 说明
format data 日期返回格式,可进行格式化修改

在.daterangepicker()中使用了回调函数function(start,end){}。

参数

名称 说明
start 获取选择日历的起始日期
end 获取选择日历的终止日期
