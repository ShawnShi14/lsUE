BSP行政区划

  • HTML
				


    


				
  var options = {
	url: "../../service/bsp/cant/getChild?code=",
	onclose: function(result) {
	    console.log(result);
	},
	multiselect: true
  }
  $("#cityBsp").proCityBsp(options);//调用proCityBsp()接口，显示地区选择

需要引用的css文件：

bootstrap.css，ui.css,form.css

需要引用的js文件：

jquery.js，ui.js，form.js


说明

使用行政区划组件，整体结构上须引用.ue-relative-layout类 。


名称 类型 默认值 说明
url string --- 数据请求地址
onclose function --- 行政区划组件关闭时触发的函数
multiselect boolean false 是否支持选中多个地区

BSP行政区划组件选中地区信息保存在input的data-code,data-name和data-type属性中
