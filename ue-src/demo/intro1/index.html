js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js，bootstrap.js，intro.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS（非模块化加载）
				
$(function() {
    introJs().setOptions({
        'nextLabel':'下一步 ',
        'skipLabel':'跳过',
        'doneLabel':'完成',
        'showStepNumbers': false,
        'showBullets': false
    }).start();
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，intro

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js非模块化加载)
				
require(['jquery','form/intro'], function($){
    $(function() {
        introJs().setOptions({
            'nextLabel':'下一步 ',
            'skipLabel':'跳过',
            'doneLabel':'完成',
            'showStepNumbers': false,
            'showBullets': false
        }).start();
    });
});

本插件须引用的css文件:

bootstrap.css，ui.css，intro.css

说明

data-step：设置引导的第几步；

data-intro：设置弹出框框里面的内容，内容可以是文本也可以是DOM结构。本实例里是DOM结构：step-header：设置弹出框的标题，是放在data-intro的DIV元素。 step-text：设置文本内容，是放在data-intro的span元素。<hr>设置弹出框里的水平线，也是放在data-intro的元素

data-position：设置弹框处于提示框的什么位置；


字段 类型 默认值 说明
nextLabel String Next 下一步按钮标签
prevLabel String Back 返回按钮标签
skipLabel Siring Skip 跳过按钮标签
doneLabel String Done 完成按钮标签
exitOnEsc Boolean true 是否点击Esc键关闭导航
exitOnOverlayClick Boolean true 是否点击遮罩区关闭导航
showStepNumbers Boolean true 是否显示提示导航的步骤
overlayOpacity number 0.8 遮罩透明度
showButtons Boolean true 是否显示按钮
showBullets Boolean false 是否显示进度
