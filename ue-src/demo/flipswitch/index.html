js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js，form.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS(js非模块化加载)
				
$(function () {
    $('.ue-flipswitch').flipswitch(callback);
    function callback(status) {
        console.log("开关状态:" + status);
    }
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，form/flipswitch

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
				
require(['jquery','form/flipswitch'], function($){
    $(function () {
        $('.ue-flipswitch').flipswitch(callback);
        function callback(status) {
            console.log("开关状态:" + status);
        }
    });
});

说明

无论是模块化加载还是非模块化加载都需要引用的css文件：

form.css

1、DOM结构及CSS样式。

首先在页面中定义一个复选框。引用.ue-flipswitch、.flipswitch-xs、 .ue-flipswitch-primary等样式。其中必须使用.ue-flipswitch，其他样式 说明如下：

    <1>.flipswitch-xs、.flipswitch-sm、.flipswitch-lg是 控制开关按钮大小的样式设置，分别是更小，小，更大，不设置任何属性时是按钮显示大小程度为大。

    <2>.ue-flipswitch-primary、.ue-flipswitch-success、.ue-flipswitch-danger .ue-flipswitch-info、.ue-flipswitch-warning用来设置开关按钮的类型，分别是，主要、成功、危险、 信息、警告类型。

    <3>使用checked属性默认当前是打开状态，disabled为禁用状态
