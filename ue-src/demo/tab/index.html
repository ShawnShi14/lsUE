$('#myTab a').click(function(e) {
    e.preventDefault();
    $(this).tab('show');
});

本插件须引用的css文件:

bootstrap.css，ui.css

本插件须引用的js文件:

jquery.js，bootstrap.js


说明

tab组件结构分为两部分：一部分是ul 列表，另一部分是点击列表的某一元素出现的对应列表里的内容。

为页面元素指定data-toggle="tab" 或 data-toggle="pill"属性激活标签页。 为ul添加.nav和.nav-tabs添加标签页样式。除此之外，应当为标签页指定data-target 或 href用以标记DOM树中的目标容器节点，激活某个标签页和对应的内容容器。
