说明

引用侧边栏组件有两种加载菜单内容的方式。第一种是在本页面编写菜单的DOM结构然后初始化时通过参数name 参数找到菜单内容；第二种是通过参数source加载页面，这种方法既可以直接加载另外一个页面，也可以通过函数来实现。

参数 类型 默认值 说明
name String sidr 菜单栏的ID名
speed string 200 执行效果的速度
side string left 出现菜单的方向
source string null 菜单的来源
displace boolean true 是否覆盖页面
onOpen function null 回调还数
onClose function null 回调函数
