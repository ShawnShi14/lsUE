js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js，jquery-ui.js，ui.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS(js非模块化加载)
				
$(".left-area").dragResize({
    target: ".right-area", 
    handles: "e" 
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，jquery-ui，ui

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
				
require(['jquery','jquery-ui','ui','demo/prettify'], function($){
    $(document).ready(function(){
        $(".left-area").dragResize({
            target: ".right-area",//被动变化元素，可为空
            handles: "e"//支持e、s、n、w四个方向
        });
    });
});

需要引用的css文件：

ui.css，jquery-ui.css


说明

名称 类型 默认值 说明
target String null 目标元素
handles String e 拖拽的方向，n s代表上下拖拽， e w代表左右拖拽
