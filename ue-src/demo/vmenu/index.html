js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js，form.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS(js非模块化加载)
					
$(function() {
    $('.ue-vmenu').vmenu({
        autostart: 1,
        autohide: 0
    });
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，form/vmenu

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
					
require(['jquery','form/vmenu'], function($){
    $(function() {
        $('.ue-vmenu').vmenu({
            autostart: 1,
            autohide: 0
        });
    });
});

组件须引用的css文件: bootstrap.css，font-awesome.css， form.css，ui.css



说明

将垂直菜单导航使用.ue-vmenu控制插件外部样式。

将垂直菜单导航使用ul 结构，ul里面是子列表li， 然后接着可以在子列表里再写ul 结构

<a>标签中data-role="leaf"表示该内容放置在叶子容器中。


名称 类型 默认值 说明
speed number 220 设置下拉菜单速度
autostart number/boolean 1/true 页面加载完毕后是否展开所有节点
autohide number/boolean 1/true 当打开兄弟节点时本节点是否自动隐藏
