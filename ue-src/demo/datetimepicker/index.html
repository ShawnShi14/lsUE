$('#dateDemo').datetimepicker({
    container: $("#create-date"),
    language: "zh-CN",
    autoclose: 1,
    startView: 2,
    minView: 2,
    format: "yyyy-mm-dd"
});

$('#timeDemo').datetimepicker({
    container: $('#create-time'),
    language: "zh-CN",
    autoclose: 1,
    startView: 1,
    minView: 0,
    maxView: 1,
    format: "hh:ii"
    //时间选择器重新初始化时，避免初始日期是1899年12月31日，而是当前日期
    forceParse: 0 
});

$('#datetimeDemo').datetimepicker({
    container: $('#create-datetime'),
    language: "zh-CN"
    autoclose: 1,
    format: "yyyy-mm-dd hh:ii"
});

需要引用的css文件：

bootstrap.css，font-awesome.css，form.css，ui.css

需要引用的js文件：

jquery.js，bootstrap.js，form.js，ui.js


说明

为了使时间选择器的格式美观，统一将时间选择器插件放在.form-group的div中。

用.ue-form调整文本框格式。


名称 类型 默认值 说明
container 容器，日期时间选择器弹出的位置相对的父元素
language String en 语言选择
autoclose Boolean false 1时,选择一个日期后关闭日期时间选择器,0则不关闭
startView Number, String 2, 'month' 日期时间选择器打开之后首先显示的视图，有2,1,0三种,2代表日期视图,1代表小时视图,0代表分钟视图
minView Number, String 0, 'hour' 日期时间选择器所能够提供的最精确的时间选择视图，2,1,0三种,表示最小到达日期视图,此处为分钟视图
maxView Number, String 4, 'decade' 日期时间选择器最高能展示的选择范围视图
weekStart Integer 0 一周从哪一天开始。0（星期日）到6（星期六）
format String 'mm/dd/yyyy' 日期格式， p, P, h, hh, i, ii, s, ss, d, dd, m, mm, M, MM, yy, yyyy 的任意组合
forceParse Boolean true 当选择器关闭的时候，是否强制解析输入框中的值。

方法

在使用插件过程中，可能会用到初始化日期时间选择器的方法：.datetimepicker(options)

方法 参数 说明 示例
remove None 移除日期时间选择器。同时移除已经绑定的event、内部绑定的对象和HTML元素 $().datetimepicker('remove');
show None 显示日期时间选择器 $().datetimepicker('show');
hide None 隐藏日期时间选择器 $().datetimepicker('hide');
update None 使用当前输入框中的值更新日期时间选择器 $().datetimepicker('update');
setStartDate startDate (String) 给日期时间选择器设置一个新的起始日期 $().datetimepicker('setStartDate', '2012-01-01');
setEndDate endDate (String) 给日期时间选择器设置结束日期 <$().datetimepicker('setEndDate', '2012-01-01');/td>
