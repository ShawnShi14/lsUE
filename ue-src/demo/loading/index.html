显示加载中效果(线形)
带文本查看的加载效果
				
function showLLoading() {
    var $t = $('#loading-container');
    $t.loading({
        isShowMask: true,
        lines: 10,
        length: 8,
        width: 5,
        radius: 15
    });
    setTimeout(function() {$t.unloading();}, 1000);
}
function showTLoading(){
    var $t = $('#loading-container');
    $t.loading({
        isShowMask: true,
        lines: 8,
        length: 0,
        width: 10,
        radius: 15,
        maskOpacity: '0.4',
        loadingText: '加载中...'
    });
    setTimeout(function(){$t.unloading();},1000)
}

需要引用的css文件：

bootstrap.css，form.css，ui.css

需要引用的js文件：

jquery.js，bootstrap.js，form.js

若使用国际化，英文请引用i18n/en_US.js，中文请引用i18n/zh_CN.js，默认为中文。


说明

使用加载中组件，首先在页面中有个触发事件的DOM结构，然后进行初始化。

setTimeout()方法设置加载时间。


名称 类型 默认值 说明
isShowMask boolean false 是否显示遮罩
lines number 12 线条数量
length number 7 线条长度
width number 4 线条宽度
radius number 10 设置加载图案的半径值
maskColor String "#000" 遮罩颜色
maskOpacity String "0.2" 遮罩透明度
color String "#fff" 花瓣颜色
opacity String "0.2" 花瓣透明度
loadingText String "加载中..." 显示文本
