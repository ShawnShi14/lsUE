说明

可以通过 data 属性或者通过 JavaScript 来使用附加导航插件,本例通过javascript来实现。

data-spy="scroll"：监听元素的滚动事件。这个元素通常都是body元素

data-target：添加需要监听的目标元素。

须为导航条中的链接指定目标id。例如，<a href="#home">home </a >必须和dom中类似<section id="home"></section>的页面元素相呼应。

名称 类型 默认值 说明
offset number 10 计算滚动位置时相对于顶部或底部的偏移量

其中：

top表示相对顶部的偏移量。

bottom表示相对底部的偏移量。
