本插件不需要写html，主要调用.pageloading()接口，实现页面加载的效果。

				
$.pageloading();

需要引用的css文件：

bootstrap.css，form.css，ui.css

需要引用的js文件及加载顺序：

jquery.js，bootstrap.js，form.js


参数说明

名称 类型 默认值 说明
opacity number 1 loading页面透明度
backgroundColor String "#fff" loading页面背景色
loadingTips String "加载中..." 提示文本
TipsColor String "#666" 提示颜色
delayTime String 1000 页面加载完成后，加载页面渐出速度
zindex number 999 loading页面层次
sleep number 0 设置挂起,等于0时则无需挂起
