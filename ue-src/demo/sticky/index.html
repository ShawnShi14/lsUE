js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，form/sticky

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
				
require(['jquery','form/sticky'], function($){
    $("#center").click(function(event) {
        $.sticky(
         "保存成功",
         {
             style : 'success',
             autoclose : 1000,
             position : 'center'
        });
    });
 
    $("#bottomright").click(function(event) {
        $.sticky(
        "警告：这是显示内容这是显 ",
        {
             style : 'warning',
             autoclose : false,
             position : 'bottom-right'
        });
    });
});

需要引用的css文件：

form.css


参数说明

名称 类型 可取值 说明
style String success, error, or warning 提示框的种类
speed String or integer fast, slow, or integer 提示框弹出的快慢程度
autoclose boolean or integer integer or false 提示框是否自动退出或者自动退出的快慢程度
duplicates boolean true or false 是否允许在同一位置下同时弹出多个提示框
position String top, bottom-right, center等 弹出的位置
