选项

名称 类型 默认值 说明
url path null 列表数据文件路径（URL） | 对象（值为 JSON 的结构，参照自定义数据结构）
selects 数组 [] 下拉选框组。输入 select 的 className,注意顺序
nodata null 子集无数据时 select 的状态。可设置为："none"(display:none), "hidden"(visibility:hidden)
required boolean false 是否为必选。设为 flase 时，会在列表头部添加 <option value="0">请选择</option>选项。
firstTitle 字符串 '请选择' 选框第一个项目的标题。如果要定义每个选框的第一个项目标题，可以通过 <select> 的 data-first-title 属性来设置。（仅在 required 为 false 时有效）
firstValue 字符串 '0' 选框第一个项目的值。如果要定义每个选框的第一个项目值，可以通过 <select>的 data-first-value 属性来设置。（仅在 required 为 false 时有效）
