jQuery返回顶部插件jQuery.toTop演示

简介

jQuery.toTop 是一款轻量级的 jQuery 返回顶部插件，压缩后不到 1KB。同时它也是可定制的，你可以设置是否自动隐藏、滚动时间、位置等等。

使用方法

  • HTML

js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js 、form.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS(js非模块化加载)
				
$('.ue-to-top').toTop({
    offset: 200
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，totop/jquery.totop

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
				
require(['jquery','totop/jquery.totop'], function($){
    $(function() {
        $('.ue-to-top').toTop({
            offset: 200
        });
    });
});

配置

属性/方法 类型 默认值 说明
autohide 布尔值 true 自动隐藏
offset 整数 420 距离顶部多少距离时自动隐藏按钮
speed 整数 500 滚动持续时间
position 布尔值 true 如果设置为 false，则需要手动在 css 中设置“按钮”的位置
right 整数 15 右侧距离
bottom 整数 30 底部距离
