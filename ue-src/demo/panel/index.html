js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，jquery-ui，form/penal，ui

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
				
require(['jquery','jquery-ui','form/panel','ui'], function($){
    $(document).ready(function() {
        $.initPanel();
      //$.initPanel.destory();
        $(".container").dragExchange({
            dragElement: ".ue-panel",//可拖拽元素
            cancel: "",//不可拖拽元素
            overClass: "overClass",//鼠标滑过样式，默认为overClass
            callback: function(a,b){
                //回调函数，a为拖拽元素，b为drop target元素
                //alert(a);
            }
        });
    });
});

本插件须引用的css文件:

bootstrap.css，ui.css，font-awesome.css

说明

将panel组件放在.ue-panel中，可以使插件样式美观，图标的位置由.ue-panel-tools设置。可实现panel面板的放大、还原以及关闭的功能。

本插件除了要引用必须的js文件外，还需要调用$(".container").dragExchange()以及$.initPanel()方法，分别用于拖拽交换与panel中工具事件初始化。

除此之外，提供了$.initPanel.destory()方法可用来销毁panel中工具触发事件。如需要禁止拖拽，可在dragExchange()的方法中设置cancel参数


名称 类型 默认值 说明
onrefresh Function null 刷新功能时执行事件
onsetting Function null 设置功能时执行事件
dragElement Object ue-panel 设置拖动交换的选择部分
cancel Object null 设置不允许拖动交换的选择部分，可用于禁止拖拽功能
callback Function a,b 传回的两个参数，为拖拽与被拖拽进行交换的目标
