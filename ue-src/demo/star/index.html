js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js，form.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS(js非模块化加载)
					
$(function(){
    $('#demo').raty({
        score: 3,
        hints: ['1分', '2分', '3分', '4分', '5分'],
        click: function(score) {
            alert('ID: ' + this.id + "\nscore: " + score);
        }
    });
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，from/star

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
					
require(['jquery','from/star'], function($){
    $(function() {
        $('#demo').raty({
            score: 3,
            hints: ['1分', '2分', '3分', '4分', '5分'],
            click: function(score) {
                alert('ID: ' + this.id + "\nscore: " + score);
            }
        });
    });
});

参数说明

名称 类型 默认值 说明
click function undefined 监听单击星星时触发的回调函数
half Boolean false 是否允许单击获得半颗星
halfShow Boolean true 是否显示半颗星
hints Array ['bad', 'poor', 'regular', 'good', 'gorgeous'] 鼠标悬浮在星星上方时依次显示的信息
number Number 5 设置显示的星星个数
numberMax Number 20 可被创建最大星星数量
readOnly Boolean false 是否只读
score Number/function undefined 设置被选中星星的个数

Callbacks

方法 说明
$('#demo').raty('score'); 获取当前分数
$('#demo').raty('score', number); 设置分数
$('#demo').raty('readOnly', boolean); 修改是否只读
$('#demo').raty('set', { option: value }); 重新设置插件新选项
$('#demo').raty('destroy'); 销毁所有的设置
