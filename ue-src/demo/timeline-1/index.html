JS非模块化应用：

需要引用的css文件：

bootstrap.css，form.css，font-awesome.css

需要引用的js文件：

jquery.js，bootstrap.js，form.js

说明

特别提醒：奇数时要添加.odd-content样式，目的是放在对时间轴的节点进行区分。

  • 非模块化加载JS

$(document).ready(function(){
    $(".timeline.animated").uTimeline();
});

JS模块化应用：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，form/utimeline

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • HTML
				
require(['jquery','form/utimeline','demo/prettify'], function($){
    $(document).ready(function(){
        $(".timeline.animated").uTimeline();
    });
});
