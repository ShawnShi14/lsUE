js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js，form.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS(js非模块化加载)
$(function () {
    $('.demo1').getNavBar(['选择镜像', '高级设置', '设置完毕']);
    $('.demo2').getNavBar(['选择镜像', '高级设置', '设置完毕']).next();
    $('.demo3').getNavBar(['选择镜像', '高级设置', '设置完毕']).next().next();
    $('.demo4').getNavBar(['选择镜像', '高级设置', '设置完毕']).next().next().prev();
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，form/breadcrumb

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
			
require(['jquery','from/breadcrumb'], function($){
    $(function () {
        $('.demo1').getNavBar(['选择镜像', '高级设置', '设置完毕']);
        $('.demo2').getNavBar(['选择镜像', '高级设置', '设置完毕']).next();
        $('.demo3').getNavBar(['选择镜像', '高级设置', '设置完毕']).next().next();
        $('.demo4').getNavBar(['选择镜像', '高级设置', '设置完毕']).next().next().prev();
    });
});

本插件须引用的css文件:

bootstrap.css，form.css


说明

在DOM结构中，用户可以根据自己的需求定义容器和类标签。设计好容器之后就可以 在js中调用getNavBar()方法，参数是要展示的步骤，放在数组中传递。其中前三个是步骤一道步骤三的的展示， 最后一个是返回步骤二的展示。


方法

方法 说明
next() 进行下一个步骤
prev() 返回前一步骤
