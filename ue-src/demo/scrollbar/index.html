js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js，bootstrap.js，form.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS(js非模块化加载)
				
$(function() {
    $('#scrollWrap').scrollbar({
        // 初始数值
        value: 30,
        // 最小数值
        minValue: 20,
        // 最大数值
        maxValue: 100,
        // 关联域
        bindField: "#memory",
        // 提示信息显示位置
        errorTip: 'right',
        // 值改变回调
        callback: function(value) {
        }
    });
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，form/scrollbar

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
				
require(['jquery','form/scrollbar'], function($){
    $(function() {
        $('#scrollWrap').scrollbar({
            // 初始数值
            value: 30,
            // 最小数值
            minValue: 20,
            // 最大数值
            maxValue: 100,
            // 关联域
            bindField: "#memory",
            // 提示信息显示位置
            errorTip: 'right',
            // 值改变回调
            callback: function(value) {
            }
        });
    });
});

需要引用的css文件：

ui.css，form.css


参数说明

名称 类型 默认值 说明
value Number 20 滑块初始值。
minValue Number 0 滑块可拖动获取到的最小值。
maxValue Number 100 滑块可拖动获取到的最大值。
bindField String "" 关联域，滑块拖动获取到的数据所关联显示到的input输入框选择器，也可以反向在该关联域里输入数值改变滑块位置，例如"#demo"。
errorTip String "top" 校验信息相对于关联域显示的位置，还可以选“left”、“right”和“bottom”。
callback function NULL 回调函数，可返回滑块滑动到位置所对应的值value。
