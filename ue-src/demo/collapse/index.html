js非模块化加载：

需要引用的js文件及加载顺序：

jquery.js，form.js，ui.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS(js非模块化加载)
			
$(function() {
    $('.ue-collapse').collapse();
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，form/collapse

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS(js模块化加载)
			
require(['jquery','form/collapse','ui'], function($){
    $(function() {
        $('.ue-collapse').collapse();
    });
});

说明

使用手风琴须采用ul li 结构，ul上引用.ue-collapse样式。

li 结构上内容分两部分：一部分是标题须引用.ue-collapse-header样式；一部分是内容， 手风琴中内容可以是列表也可以是文本，如果是列表须引用.ue-collapse-body样式，如果是文本须同时引用.ue-collapse-body样式和.text样式


方法 类型 说明
.collapse() boolean 只允许一个选项卡打开, 默认处于闭合状态
.collapse(false) boolean 只允许一个选项卡打开, 默认处于闭合状态
.collapse(true) boolean 支持同时展开多个选项卡, 默认处于闭合状态
.collapse(0) 整型 默认展开第一个选项卡,只允许一个选项卡展开
.collapse([1,2]) 数组 默认展开第二个和第三个选项卡,支持多个选项卡同时展开
