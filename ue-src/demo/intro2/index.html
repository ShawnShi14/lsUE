js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，intro

说明

introjs-active：引用导航组件需引用此整体样式

通过设置id="step"：控制对哪部分内容进行导航

.introjs-active .introjs-skipbutton, .introjs-active .introjs-skipbutton:focus: 设置跳过按钮的背景图

.introjs-active .introjs-skipbutton:hover ： 设置跳过按钮悬浮时的背景图

.introjs-active .introjs-nextbutton, .introjs-active .introjs-nextbutton:focus： 设置下一步按钮的背景图

.introjs-active .introjs-nextbutton:hover： 设置下一步按钮悬浮时的背景图

element: '#step'：获取导航的内容

intro：设置导航组件的引导图片

position:：导航组件的引导图片的方向


参数 类型 默认值 说明
nextLabel String Next 下一步按钮的文本内容
prevLabel String Back 返回按钮的文本内容
skipLabel Siring Skip 跳过按钮的文本内容
doneLabel String Done 完成按钮的文本内容
exitOnEsc Boolean true 是否点击Esc键关闭导航
exitOnOverlayClick Boolean true 是否点击遮罩区关闭导航
showStepNumbers Boolean true 是否显示提示导航的步骤
overlayOpacity number 0.8 遮罩透明度
showButtons Boolean true 是否显示按钮
showBullets Boolean false 是否显示进度
