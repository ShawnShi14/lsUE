 $('.ue-unslider').unslider({
     fluid: true,
     dots: true,
     speed: 500
 });
 var unslider = $('.ue-unslider').unslider();
  $('.ue-unslider-arrow').click(function() {
     var fn = this.className.split(' ')[1];
     //  Either do unslider.data('unslider').next() or .prev() depending on the className
     unslider.data('unslider')[fn]();
 });

需要引用的css文件：

bootstrap.css，font-awesome，form.css，ui.css

需要引用的js文件：

jquery.js，form.js



说明

组件引用.ue-relative-layout整体样式,或者另一个声明了 position: relative;的元素，然后加入图片轮播的 HTML 代码

.ue-unslider设置轮播图片样式，添加向前向后翻的链接.ue-unslider-arrow prev.ue-unslider-arrow next，同时还可以设置前后翻页按钮的样式。


名称 类型 默认值 说明
speed 整型或布尔型 500 动画的速度,没有过度效果时为 false
delay 整型或布尔型 3e3 幻灯片之间的延迟，没有自动播放时为false
keys 布尔型 true 允许键盘左右键控制
dots 布尔型 false 显示点导航
fluid 布尔型 false 支持响应式设计

回调函数

名称 说明
complete 播放每张幻灯片后调用的函数
