js非模块化加载：

需要引用的js文件及顺序：

jquery.js，form.js

引用方式：

正常js引用方式 <script type="text/javascript" src="jquery.js路径"></script>

  • JS（js非模块化加载）
				
$(document).ready(function(){
    $("#ue-content").colorpicker(function(el,color){
        $(el).css("backgroundColor",color);
    });
});

js模块化加载：

在require之前需要引用的js文件：

require.js，main.js

引用方式：

<script src="require.js路径"></script>

<script src="main.js路径"></script>

之后在require中加载组件所需要的js文件，jquery，form/colorpicker

将js文件放在数组里面，js的引用在非模块化中是无序的，注意文件不要加.js后缀，因为RequireJS会自动加载。

  • JS（js模块化加载）
				
require(['jquery','bootstrap','from/colorpicker'], function($) {
    $(function() {
        $("#ue-content").colorpicker(function(el,color){
            $(el).css("backgroundColor",color);
        });
    });
});

说明：

不论是模块化加载还是非模块化加载本组件必须引用的css文件：

form.css
